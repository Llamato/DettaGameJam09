## Development

Some tools are good for consistent editing in the team.
Some editors respect the provided configuration files out of the box.
These tool can also usually be installed via package managers, e.g. `brew`
and then used in Terminal.
For this repository following tools might be useful:

- `editorconfig`
- `editorconfig-checker`
- `fourmolu`
- `jspcd`
- `prettier`
- `yamllint`
